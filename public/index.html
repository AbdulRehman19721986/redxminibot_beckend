How to Deploy
1
Fork the repo — github.com/AbdulRehman19721986/REDXBOT-MD
2
Deploy to Railway / Render / Heroku — connect GitHub, set env vars
3
Set required env vars: OWNER_NUMBER, BOT_NAME, PREFIX, MENU_IMAGE
4
Visit your bot URL — enter phone number, get pairing code
5
Enter code in WhatsApp → Linked Devices → Link a Device
6
Type .menu — select a category number. Type .allmenu for all 500+ commands
7
Admin Panel — visit /admin on your bot URL (credentials in your env)
