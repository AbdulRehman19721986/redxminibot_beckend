## 🚀 Quick Deploy

### Railway (Recommended)
1. Fork or upload this repo to GitHub
2. Create new Railway project → Deploy from GitHub
3. Set environment variables (see below)
4. Open your Railway URL → Pair your number

### Heroku
```
git init && git add . && git commit -m "REDXBOT302 v7"
heroku create your-app-name
heroku config:set BOT_NAME="🔥 REDXBOT302" OWNER_NUMBER=923009842133
git push heroku main
```

### Local
```bash
npm install
cp .env.example .env
# Edit .env with your values
npm start
```

## 📱 How to Pair

1. Open your bot URL in browser
2. Enter your WhatsApp number (e.g. `923001234567`)
3. Click **Get Pairing Code**
4. Go to WhatsApp → Settings → Linked Devices → Link a Device → Link with phone number
5. Enter the 8-digit code
6. ✅ Bot connected! Check WhatsApp for your **Deploy Key**
