## ⚡ Quick Start (No Session ID Needed!)

```bash
git clone <your-repo>
cd backend
cp .env.example .env
# Edit .env — set PAIRING_NUMBER to your WhatsApp number
npm install
npm start
```

Open the **frontend** URL → Enter your number → Get code → Enter in WhatsApp Linked Devices → Done! 🎉
